# Análisis de Evasión de Clientes (Churn) - TelecomX

## Propósito del Proyecto
El objetivo de este proyecto es analizar la **evasión de clientes (Churn)** de la empresa ficticia **TelecomX**. Se busca identificar patrones y factores que influyen en la cancelación del servicio, utilizando datos de clientes, servicios contratados y cargos asociados.  
Los resultados permiten diseñar estrategias para **mejorar la retención de clientes** y reducir la evasión.
